# jeetmantra2026

JeetMantra is a multi-role education platform with student, teacher, partner, admin, superadmin, institute, branch, and school dashboards.

## Available screen references

The repository includes prototype UI files for the key screens, which should be checked against the provided design screenshots:

- `JeetMantra-AllRoles.html` — role switcher and combined portal overview
- `JeetMantra-Dashboard.html` — student dashboard flows
- `JeetMantra-TeacherDashboard.html` — teacher dashboard and attendance workflows
- `JeetMantra-PartnerDashboard.html` — partner services and booking flows
- `JeetMantra-Website.html` — public marketing website and program listings
